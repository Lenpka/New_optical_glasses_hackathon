# Инструмент для работы с составами оптических стекол

Простой интерфейс для **внешнего пользователя**: восстановить состав по свойствам или сгенерировать кандидаты нейросетью.

## Что умеет

| Режим | Метод | Вход | Выход |
|--------|--------|------|--------|
| **Восстановление** | Поиск соседей SciGlass (v3) | n_d, ν_d, (ρ, T_g) или марка SCHOTT | Реальный состав ближайшего стекла + таблица соседей |
| **Генерация** | cWGAN-GP + surrogate | Целевые n_d, ν_d | Список новых кандидатов с оценкой distance |

Важно: восстановление **не синтезирует** стекло — только находит ближайшие записи в базе SciGlass.

## Быстрый старт

### 1. Проверка

```bash
python glass_tool.py check
```

Должны быть OK:
- `sciglass_zip` — для восстановления;
- `gan_generator`, `forward_models`, `merged_data` — для генерации.

### 2. Восстановление по свойствам

```bash
python glass_tool.py recover --nd 1.85 --vd 25 --density 4.1 --tg 520
```

Без ρ и T_g (подставятся медианы пула SciGlass):

```bash
python glass_tool.py recover --nd 1.85 --vd 25
```

### 3. Восстановление по марке SCHOTT

```bash
python glass_tool.py recover --glass "N-SF11"
python glass_tool.py list-glasses --limit 50
```

### 4. Генерация (GAN)

```bash
python glass_tool.py generate --nd 1.90 --vd 22 -n 400 --top 15 --out candidates.csv
```

### 5. Интерактивное меню (терминал)

```bash
python glass_tool.py interactive
```

### 6. Веб-интерфейс (опционально)

```bash
pip install streamlit
streamlit run glass_tool_app.py
```

## Файлы, которые должны лежать в проекте

| Файл / папка | Назначение |
|----------------|------------|
| SciGlass `select_SciGK.csv.zip` | База для восстановления |
| `schott-optical-glass-overview-excel-format-en 202501113.xlsx` | Каталог SCHOTT |
| `merged_data.parquet` | Данные для GAN / surrogate |
| `output/gan_design/checkpoints/generator.pt` | Обученный генератор |
| `output/gan_design/condition_stats.json` | Нормализация условий GAN |
| `output/inverse_design_2h/forward_models/*.joblib` | Модели n_d, ν_d, ρ, T_g |

Пути можно переопределить: `--sciglass-zip`, `--gan-dir`, `--forward-models`, `--data`.

## Как читать результат восстановления

- **primary_composition** — основной ответ (лучший сосед, 4 свойства).
- **jaccard_topk** — пересечение соседей при 2 и 4 признаках; < 0.3 — ill-posed.
- **uncertainty_score** — выше → менее уверенный ответ.
- **composition_pls** — только справочно (локальная модель), не замена соседа.

## Как читать результат генерации

- **feasible** — прошёл фильтры (PbO≈0, T_g, SiO₂, distance, …).
- **distance** — новизна; сравнивайте с порогом p95 в отчёте (~1.4).
- Высокий n_d при большом distance — риск экстраполяции surrogate.

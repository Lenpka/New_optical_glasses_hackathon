1. Самый сильный старт — каталоги оптических стекол производителей

-  url SCHOTT Optical Glass Catalog https://www.schott.com/en-gb/products/optical-glass-p1000267

-  url CDGM Optical Glass Catalog https://www.cdgmglass.com/optical-glass/ 

Что можно получить:
- ~200–400 коммерческих стекол;
- пары (n, ν);
- иногда составные семейства.

Минус: состав часто скрыт.

4. Для именно ML по стеклам — статьи с готовыми датасетами

Начни с таких направлений поиска:

- “machine learning optical glass design”
- “glass property prediction dataset”
- “inverse design of optical glasses”
- “high refractive index low dispersion glass machine learning”
- “SciGlass optical property prediction”

Очень часто авторы выкладывают CSV.

Дальше я бы строил не просто регрессию состава → n.

Лучше сформулировать как многокритериальную оптимизацию:

Цель:
- maximize n
- maximize ν
- minimize ρ
- maximize transmission
- minimize PbO

Ограничения:
- Tg > 500–600 °C
- отсутствие кристаллизации
- сумма компонентов = 100 mol%

То есть задача ближе к:
composition → surrogate model → Pareto front → inverse design

Для признаков сразу заложить не только mol% оксидов, но и физико-химические дескрипторы:

- средний ионный радиус;
- поляризуемость;
- оптическая основность;
- средняя электроотрицательность;
- field strength Z/r²;
- packing density;
- концентрация NBO;
- молярный объём.

Для хакатона это уже выглядит как полноценный pipeline исследования, а не просто подбор модели.

Для твоей задачи (ML + дизайн составов) не скачивай всё подряд. На странице SCHOTT бери именно данные по маркам стекол и свойствам.

Смотри в таком порядке:

1. **Optical Glass Pocket Catalog / Optical Glass Catalog (PDF)** ← начать отсюда  
Что даст:
- список стекол;
- показатель преломления **n_d**;
- число Аббе **ν_d**;
- плотность;
- температурные характеристики;
- иногда коэффициенты расширения и пропускание.

Это уже можно превратить в таблицу:
```
Glass | nd | vd | density | Tg | CTE ...
```

2. **TIE (Technical Information / Technical Information Sheets)** ← самое полезное дальше  
Обычно там:
- формулы дисперсии (Sellmeier);
- спектральное пропускание;
- dn/dT;
- детали измерений.

Это уже признаки для более серьёзной модели.

3. **Transmission / Environmental / Product tables** — только если останется время  
Для хакатона вторично.

4. Не трать время на:
- сертификаты;
- маркетинговые буклеты;
- CAD;
- пресс-релизы.

Практически твой план такой:

Шаг 1 — скачать **Catalog PDF**  
Шаг 2 — выгрузить таблицу стекол  
Шаг 3 — построить график:
- X = ν_d (число Аббе)
- Y = n_d (показатель преломления)

Получишь классическую **диаграмму Аббе**.

Потом выделить область:
```
n_d > 1.75
ν_d > 30
без PbO
```
— это и будет область поиска перспективных составов.

И ещё важная мысль: у SCHOTT составы почти всегда скрыты. Поэтому SCHOTT — это источник **целевых свойств**, а не датасет для inverse design. Составы потом добираются из SciGlass / INTERGLAD / статей.
